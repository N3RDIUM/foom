# `foom`

A curses-based mpd client.

## Capabilities

- [ ] Search for music on YouTube
- [ ] Download music from YouTube
- [ ] Parallely download multiple songs
- [ ] Create and manage playlists
- [ ] Play music
- [ ] Visualize music
- [ ] FULL Mpris integration
